# Projects

CLI para crear proyectos de Node.js, Python, Go y Bash con las frameworks más populares.

## Instalación

```bash
uv sync
```

## Uso

```bash
uv run main.py
```

## Frameworks disponibles

| Framework | Comando |
|-----------|---------|
| Astro     | Proyecto con soporte Tailwind y Node adapter |
| Vite      | Proyecto con soporte Tailwind |
| Angular   | Proyecto Angular |
| UV        | Proyecto Python con UV |
| Rspack    | Proyecto con Rspack |
| Flowbite  | Proyecto React con Flowbite |
| Go        | Proyecto Go con go mod |
| Bun       | Proyecto JavaScript con Bun |
| Bash      | Script Bash executable |

## Dependencias

- [InquirerPy](https://github.com/kazhala/InquirerPy) - Interfaz de línea de comandos interactiva
- [Halo](https://github.com/manrajgrover/halo) - Spinner para terminal
